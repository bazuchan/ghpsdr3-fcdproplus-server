ghpsdr3-fcdproplus-server
=========================

It is Funcube Dongle Pro Plus server for ghpsdr3-alex implementation on python.

Dependencies is:
 - python-alsaaudio
 - https://github.com/bazuchan/cython-hidapi (fork of https://github.com/gbishop/cython-hidapi)
